== Description ==

Basically it just a WordPress "Text Widget" but with additional icon font selector based on [FontAwesome](http://fortawesome.github.io/Font-Awesome/). The Icon will show just before widget title.

== Installation ==

1. Upload the entire "widget-text-icon" folder to the "/wp-content/plugins/" directory
2. DO NOT change the name of the "widget-text-icon" folder
3. Activate the plugin through the "Plugins" menu in WordPress
4. Drag and drop "Widget Text Icon" into active sidebar

== Frequently Asked Questions ==
