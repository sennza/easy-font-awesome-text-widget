== Description ==

Basically is a text widget but with iconic font selector based on http://fortawesome.github.com/Font-Awesome/

== Installation ==

1. Upload the entire `widget-text-icon` folder to the `/wp-content/plugins/` directory
1. DO NOT change the name of the `widget-text-icon` folder
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Drag and drop 'Widget Text Icon' into active sidebar
